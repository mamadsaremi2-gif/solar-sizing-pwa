# Solar Sizing PWA

این بسته، فایل مهندسی JSX شما را داخل یک پوسته PWA برای Vite قرار می دهد.

## اجرا

```bash
npm install
npm run dev
```

## ساخت نسخه نهایی

```bash
npm run build
npm run preview
```

## یادداشت مهم

این بسته فرض می کند کامپوننت های shadcn/ui مورد استفاده در فایل اصلی شما داخل مسیر `src/components/ui` موجود هستند.
اگر در پروژه فعلی آن ها را ندارید، باید آن کامپوننت ها را نیز به پروژه اضافه کنید یا فایل اصلی را به نسخه بدون shadcn تبدیل کنید.

## قابلیت های PWA

- نصب روی موبایل و دسکتاپ
- manifest
- service worker
- صفحه آفلاین
- نوار وضعیت PWA
